# mcPHASES: activity, sleep, physiology → next-day symptoms

Code and notebooks for predicting **next-day** self-reported symptom severity (cramps, fatigue, mood; 0–5 scales) from **prior-day** activity, **prior-night** sleep architecture, and **overnight** physiology (HRV, resting HR, skin temperature), with evaluation of **cycle phase** as baseline and as a moderator of model performance.
